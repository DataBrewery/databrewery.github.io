Data Brewery

Data Brewery is a set of Python frameworks with tools for data analysis.

Aggregated data browsing, reporting and multidimensional modeling. Contains set of tools, OLAP HTTP server and light-weight Python framework.

Overview Download Documentation Models Extensions

Flow based data processing with support for variety of data sources, data targets. Suitable for extraction, transformation and loading (ETL) as well as data auditing and data quality monitoring.
